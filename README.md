## Ethernaut Solutions

Solutions for Ethernaut starting from CoinFlip

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```

### Gas Snapshots

```shell
$ forge snapshot
```

### Anvil

```shell
$ anvil
```

### Deploy

```shell
$ forge script src/contracts/<contract>/<contract>.sol:Contract --rpc-url <your_rpc_url> --private-key <your_private_key>
```
